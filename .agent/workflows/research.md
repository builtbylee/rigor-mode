# Workflow: research

## Purpose
Produce reliable research output without hallucinations.

## Steps
1. Define the exact question.
2. Enumerate what you already know vs need to learn.
3. Identify sources (docs > code > reputable posts).
4. Extract key facts with citations/quotes.
5. Summarize findings + recommended action.

## Output
- Question
- Sources
- Findings
- Recommendation
- Open questions
